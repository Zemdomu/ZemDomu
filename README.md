# ZemDomu

Semantic HTML linter for HTML, JSX, TSX, and Vue templates. Detects accessibility,
SEO, and structure issues before deployment.

## What it is

ZemDomu Meta is the community hub for the ZemDomu ecosystem. Use this repo for
discussions, announcements, and general issues that do not belong to a specific
service.
